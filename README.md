# THUCS_CSPM

C/C++ code for THUCS course: Computer System Performance Measurement

## Setup

- c lib
  - libunwind
- c++ lib
  - cxxopts: `lib/cxxopts/cxxopts.hpp`
- clang
- make
- zig (optional)
- python3

  - polars
  - matplotlbi
- pin

- papi
  - `<WORKSPACE_ROOT>/papi/`
  - https://github.com/icl-utk-edu/papi
